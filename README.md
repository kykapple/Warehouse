### 개발 내역
- FE
  - 동네 모달창 구현
    - 로그인 상태의 경우, 사용자의 동네 출력 
    - 비로그인 상태의 경우, 동네 목록이 없으므로 모달창을 출력하지 않음
  - 동네 설정 페이지 구현

- BE
  - 사용자의 동네 정보를 조회하는 로직 구현
  - session store를 메모리 -> FileStore -> RedisStore로 변경
